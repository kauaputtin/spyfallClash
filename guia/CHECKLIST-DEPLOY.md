# ✅ Checklist - Deploy do Spyfall Clash

Use esta checklist para não esquecer nenhum passo!

## 📋 Pré-requisitos

- [ ] Node.js instalado
- [ ] NPM funcionando
- [ ] Jogo funcionando localmente (testado em `http://localhost:3000`)
- [ ] Todos os arquivos na pasta `Spyfall Clash`

## 🔧 Preparação

- [ ] Dependências instaladas (`npm install` ou `install.bat`)
- [ ] Servidor testado localmente (`node server.js` ou `start.bat`)
- [ ] Arquivo `.gitignore` criado e configurado
- [ ] Todos os arquivos verificados

## 📦 GitHub

- [ ] GitHub Desktop instalado
- [ ] Conta GitHub criada/fazer login
- [ ] GitHub Desktop conectado à conta
- [ ] Repositório local adicionado no GitHub Desktop
- [ ] Primeiro commit feito
- [ ] Repositório criado no GitHub (ou conectado a um existente)
- [ ] Código enviado para o GitHub (push)
- [ ] Arquivos visíveis no GitHub.com

## 🚀 Render

- [ ] Conta Render criada
- [ ] Login feito no Render (com GitHub)
- [ ] Novo Web Service criado
- [ ] Repositório GitHub conectado
- [ ] Configurações preenchidas:
  - [ ] Name: `spyfall-clash`
  - [ ] Environment: `Node`
  - [ ] Build Command: `npm install`
  - [ ] Start Command: `npm start`
  - [ ] Plan: `Free`
- [ ] Deploy iniciado
- [ ] Deploy concluído (status: "Live")
- [ ] URL copiada

## 🧪 Teste

- [ ] Jogo acessível pela URL do Render
- [ ] Tela inicial carrega
- [ ] Criar sala funciona
- [ ] Código de sala gerado
- [ ] Entrar em sala funciona (teste em outro navegador/dispositivo)
- [ ] Jogadores aparecem na lista
- [ ] Configuração de impostores funciona
- [ ] Iniciar partida funciona

## 🎮 Compartilhar

- [ ] URL compartilhada com amigos
- [ ] Amigos conseguem acessar
- [ ] Jogo funcionando online
